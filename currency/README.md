## 💻 Installation

After cloning the project run  the following commands in a CLI to install the library

```sh
❯ cd /path/to/workspace/3-musketeers/currency
❯ npm install # or yarn

```

## 🛠 Usage

To execute the programm then simply run:

```sh
❯ node cli.js
```
